1. Open File > Import Project... and navigate to the storm-starter directory of your storm clone 
(e.g. ~/Documents/storm/examples/storm-starter).
2. Select Import project from external model, select "Maven", and click Next.
3. In the following screen, enable the checkbox Import Maven projects automatically. 
Leave all other values at their defaults. Click Next.
4. Click Next on the following screen about selecting Maven projects to import.
5. Select the JDK to be used by IDEA for storm-starter, then click Next. (select java-6-oracle)
6. You may now optionally change the name of the project in IDEA. The default name suggested by IDEA is "storm-starter". 
Click Finish once you are done.
